# Tech Support Scripts

A collection of bash and Python scripts built during Week 1 of tech support training.

## Scripts

### Bash Scripts
- `sysinfo.sh` — displays system information (RAM, disk, IP)
- `backup.sh` — backs up files to a backup folder
- `createuser.sh` — creates and configures a new Linux user

### Python Scripts
- `techcheck.py` — monitors disk usage across multiple servers
- `ticketsystem.py` — creates, resolves and saves support tickets
- `readcsv.py` — reads and displays CSV user data
- `filtercsv.py` — filters active users from CSV
- `support_report.py` — generates support reports with email validation
- `file_organizer.py` — automatically organizes files by type with logging

## Skills Demonstrated
- Linux administration
- Python scripting
- File handling & CSV processing
- Regular expressions
- Logging & error handling
- Git version control
