# Tech Support Scripts

A collection of bash and Python scripts for tech support tasks.

## Scripts

### Bash Scripts
- `sysinfo.sh` — displays system information
- `backup.sh` — backs up files to a backup folder
- `createuser.sh` — creates and configures a new Linux user

### Python Scripts
- `techcheck.py` — monitors disk usage across servers
- `ticketsystem.py` — manages support tickets
- `readcsv.py` — reads and displays CSV user data
- `filtercsv.py` — filters active users from CSV
- `support_report.py` — generates support reports with email validation
